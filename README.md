## 🔧 Installation

1. Clone the repository

```bash
git clone https://github.com/ajay-chauhan-324/portfolio.git
```

2. Navigate to the project folder

```bash
cd portfolio
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```
